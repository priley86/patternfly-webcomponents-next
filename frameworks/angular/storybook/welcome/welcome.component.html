Welcome to Patternfly Angular + Web Component Storybook

Patternfly Angular + Web Component Storybook is a UI component dev environment for your Angular components.


A story is a single state of one or more UI components. Stories provide a visual test case demonstrating various component use cases and component interactions.

Usually we create stories with smaller UI components in the app. Have a look at the Writing Stories section in our documentation.

NOTE:
You can modify storybook/webpack.config.js to add webpack loaders and plugins you are using in this project.
